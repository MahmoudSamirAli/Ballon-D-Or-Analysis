Designed and developed an interactive Football Performance Analytics Dashboard to compare Ousmane Dembélé and Mohamed Salah’s performance and efficiency in the 2025 Ballon d’Or race. The project focused on transforming match statistics into visual insights to evaluate player efficiency, consistency, and overall contributions across different competitions and match outcomes.
Tools & Skills: Excel (Power Query, Pivot Tables, Charts, Dashboard Design, Data Visualization), Sports Analytics, Performance Comparison, KPI Design, Insight Communication.
